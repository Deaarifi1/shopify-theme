# Shopify Theme – Internship Assignment

## Overview

This repository contains a custom **Shopify theme** developed as part of an **internship application assignment**. The goal of the task was to demonstrate practical skills in Shopify theme development, Liquid templating, front-end development, and general understanding of e‑commerce user experience.

The theme is built to be clean, responsive, and easy to customize, following Shopify best practices.

## Purpose of the Project

The project was created to:

* Demonstrate proficiency with **Shopify Liquid**, **HTML**, **CSS**, and **JavaScript**
* Show understanding of Shopify theme architecture
* Build reusable and customizable sections
* Apply responsive design principles
* Follow clean and readable code practices

## Technologies Used

* **Shopify Liquid**
* **HTML5**
* **CSS3**
* **JavaScript**
* **Shopify Theme Architecture**

## Theme Structure

```
├── assets/        # CSS, JavaScript, images
├── config/        # Theme settings
├── layout/        # Main theme layout files
├── sections/      # Reusable Shopify sections
├── snippets/      # Reusable Liquid snippets
├── templates/     # Page templates
└── locales/       # Language and translation files
```

## Notes

* This project was created **for educational and evaluation purposes** as part of an internship application
* It is not intended for direct production use without further optimization and testing
